# Politique de Confidentialité – Le Date dont vous êtes le héros

**Dernière mise à jour : 6 juillet 2026**

Cette politique de confidentialité décrit comment l’application **Le Date dont vous êtes le héros** (ci-après l’« Application ») traite les données personnelles.

## 1. Responsable du traitement

- **Entité** : Les Dateurs (éditeur de l’Application)
- **Application concernée** : Le Date dont vous êtes le héros
- **Contact confidentialité** : **castagnet.cyril at gmail.com**

## 2. Données traitées

Selon votre usage, l’Application peut traiter les catégories suivantes :

- **Données de compte** : adresse e-mail (si inscription/connexion e-mail), identifiant utilisateur technique.
- **Données d’authentification** : informations de session (tokens) gérées via Supabase Auth.
- **Données de progression de jeu** : actions tentées, fins découvertes, médailles, favoris, paramètres, points/cacahuètes, tentatives.
- **Données techniques minimales** : informations nécessaires au fonctionnement de l’authentification et de la synchronisation.
- **Données stockées localement sur l’appareil** : préférences, état local de jeu, mode invité, caches.

## 3. Données non collectées / permissions sensibles

L’Application n’a pas pour objet de collecter des données sensibles telles que :

- contacts,
- SMS,
- microphone,
- caméra,
- localisation précise,
- agenda.

Si une permission sensible devait être ajoutée dans une version future, la présente politique sera mise à jour avant déploiement.

## 4. Finalités du traitement

Les données sont traitées pour :

- créer et gérer votre compte utilisateur ;
- permettre la connexion (e-mail / OAuth Google via Supabase) ;
- sauvegarder et synchroniser votre progression de jeu ;
- assurer le fonctionnement technique et la sécurité de l’Application ;
- gérer les demandes d’assistance et les obligations légales.

## 5. Bases légales (RGPD)

Selon les cas, les bases légales sont :

- **Exécution du contrat** : fournir les fonctionnalités attendues de l’Application.
- **Intérêt légitime** : sécurité, prévention d’abus, stabilité technique.
- **Obligation légale** : conformité réglementaire et demandes d’autorité compétente.
- **Consentement** : lorsqu’exigé par la réglementation (le cas échéant).

## 6. Destinataires et sous-traitants

Vos données peuvent être traitées par :

- **Supabase** (authentification, base de données, stockage technique des données de progression) ;
- **Google** (uniquement dans le cadre de la connexion Google si vous choisissez cette méthode).

Ces prestataires agissent selon leurs propres engagements contractuels et de sécurité.

## 7. Hébergement et transferts hors UE

Les données peuvent être hébergées et/ou transitent via les infrastructures des prestataires techniques (notamment Supabase et Google), y compris potentiellement hors de l’Union européenne selon leur configuration.

En cas de transfert hors UE, des mécanismes de protection appropriés sont appliqués par les prestataires (ex. clauses contractuelles types lorsque requis).

## 8. Durée de conservation

- **Compte actif** : conservation pendant la durée d’utilisation de l’Application.
- **Données de progression** : conservées tant que le compte est actif, puis supprimées ou anonymisées dans un délai raisonnable après demande ou suppression de compte, sauf obligation légale contraire.
- **Données locales sur appareil** : conservées jusqu’à suppression par l’utilisateur (désinstallation, purge locale, déconnexion selon les cas).

## 9. Sécurité

Des mesures techniques et organisationnelles raisonnables sont mises en place pour protéger les données (contrôle d’accès, sécurisation des échanges, configuration de sécurité côté backend, etc.).

Malgré ces mesures, aucun système n’est totalement invulnérable.

## 10. Vos droits

Conformément au RGPD (et selon la loi applicable), vous pouvez :

- demander l’accès à vos données ;
- demander la rectification de données inexactes ;
- demander l’effacement de vos données ;
- demander la limitation ou vous opposer à certains traitements ;
- demander la portabilité des données lorsque applicable ;
- retirer votre consentement lorsque le traitement repose sur celui-ci.

Pour exercer vos droits : **castagnet.cyril at gmail.com**.

Vous pouvez également introduire une réclamation auprès de l’autorité de contrôle compétente (ex. CNIL en France).

## 11. Comptes, suppression et mode invité

- Le **mode invité** permet un usage sans création immédiate de compte, avec stockage local et/ou synchronisation limitée selon l’état de connexion.
- En cas de demande de suppression de compte, les données associées sont supprimées dans les limites techniques et légales applicables.
- **Point de contact suppression compte** : **castagnet.cyril at gmail.com**.

## 12. Mineurs

L’Application n’est pas destinée en priorité aux jeunes enfants.  
Si vous êtes parent/tuteur et pensez qu’un mineur a fourni des données personnelles sans autorisation appropriée, contactez-nous pour traitement de la demande.

## 13. Modifications de la politique

Cette politique peut évoluer pour refléter les changements techniques, légaux ou produit.  
La date de dernière mise à jour est indiquée en en-tête. En cas de changement substantiel, une information appropriée sera fournie dans l’Application et/ou sur la fiche Play Store.
